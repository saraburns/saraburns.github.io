EmotiSphere

EmotiSphere was created for the class Tangible User Interfaces, as a way to solidify the intangible relationship between emotion and music. It reads in physicval signs of emotion and determines an emotional profile. Then, it plays an algorithmic composition designed to match the emotion of the profile. The EmotiSphere uses Arduino, Processing, and Pure Data, and was accepted as Works in Progress to the TEI 2015 Conference at Stanford University.

MIT App Inventor Concept Map

The Concept Map is an educational tool developed as part of an NSF-sponsored project on MIT App Inventor, a blocks-based mobile development program. The map is designed to visually lay out the concepts encountered in App Inventor in order to bridge the gap between programming in blocks and the CS theory behind it. I built the map independently, using primarily JavaScript and the sigma.js graphing library. The project has been presented at the MIT App Inventor Summit and will be presented at CCSCNE 2015.

Schedule Sculptor

This project was created as a final project for the course CS 230: Data Structures. It was built with Claire Schlenker. The program is written in Java, and allows users to build potential course schedules. A course recommender engine provides courses that will not conflict with the user's current schedule. Schedules can be viewed as they are constructed.
